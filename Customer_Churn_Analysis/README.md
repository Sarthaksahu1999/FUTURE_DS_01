# 📊 Customer Churn & Retention Analysis

## 📌 Project Overview
This project analyzes customer churn behavior in a telecom subscription business. The goal is to identify churn patterns, understand retention drivers, and provide actionable business insights.

---

## 🎯 Objectives
- Analyze customer churn trends
- Perform cohort-based retention analysis
- Identify high-risk customer segments
- Provide data-driven recommendations

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Excel

---

## 📊 Key Insights
- Month-to-month contract customers have the highest churn
- Customers in the first 12 months are more likely to churn
- Retention improves significantly after 24 months
- Long-term customers are more stable

---

## 📈 Dashboard Features
- KPI Cards (Total Customers, Churn Rate, Avg Tenure)
- Churn analysis by contract and tenure
- Payment method insights
- Cohort retention heatmap (Python)

---

## 💡 Business Recommendations
- Improve onboarding for new users
- Encourage long-term contracts
- Target high-risk customers early
- Build loyalty programs
