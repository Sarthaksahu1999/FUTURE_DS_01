# 📊 Marketing Funnel & Conversion Analysis

## 📌 Project Overview
This project analyzes a marketing funnel to identify user drop-offs, conversion rates, and opportunities to improve business performance.

## 🎯 Objective
To understand how users move from leads → interested → customers and identify key optimization areas.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Tableau

## 📈 Key Metrics
- Total Leads: 45,211
- Interested Users: 22,551
- Customers: 5,289
- Conversion Rate: 11.7%

## 🔍 Key Insights
- ~50% users drop at initial stage
- Only ~23% of interested users convert
- Students and retired segments show highest conversion
- Longer interaction duration improves conversion

## 💡 Recommendations
- Improve lead qualification strategy
- Focus on high-performing customer segments
- Optimize final conversion stage
- Reduce drop-offs in early funnel stages
